# NDS9 Signature

A custom HTML email signature template for NDS9, featuring a clean design with contact icons and company branding.

## Features

- Responsive email signature design
- Inline SVG icons for portability
- Copy-to-clipboard functionality
- External CSS and JS for easy customization
- Optimized for email clients like Outlook

## Files

- `index.html` - Signature preview and template
- `assets/css/styles.css` - Styling for signature and demo
- `assets/js/main.js` - JavaScript for copy functionality
- `package.json` - Scripts for local development

## Usage

1. Open `index.html` in your browser to preview
2. Customize placeholders (name, contact info, etc.)
3. Use the copy button to get email-ready HTML
4. Paste into your email client's signature settings

For local development: `npm install -g http-server && npm start`

## Notes

- Icons use color #685e56 for consistency
- Copy function inlines styles and converts icons for email compatibility
